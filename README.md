## 📁 Projeto

Clima Atual é uma aplicação web minimalista e elegante que exibe as condições meteorológicas em tempo real de Vila Velha - ES. Com uma interface limpa e moderna, o projeto atualiza automaticamente os dados climáticos a cada 15 segundos, proporcionando informações sempre atualizadas sobre temperatura e condições do tempo. A aplicação apresenta um design responsivo com fundo escuro e tipografia clara, além de ícones visuais representativos das condições climáticas atuais.

## ✨ Funcionalidades

- 🌡️ **Temperatura em tempo real** - Exibição da temperatura atual em Celsius
- 🔄 **Atualização automática** - Dados atualizados a cada 15 segundos
- 🎨 **Interface responsiva** - Funciona perfeitamente em desktop e mobile
- 🌦️ **Ícones climáticos** - Representação visual das condições do tempo
- 🌍 **Localização específica** - Focado em Vila Velha, Espírito Santo
- ⚡ **Performance otimizada** - Carregamento rápido e leve
